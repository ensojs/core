## Usage

Utility for setting up Enso projects. There is no need to install this globally.

```bash
# npx (default)
npx @ensojs/create-app && cd enso-app && yarn dev

# yarn (default)
yarn create @ensojs/app && cd enso-app && yarn dev
```
